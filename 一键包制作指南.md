# 白嫖大师 - 一键包制作指南

## 什么是一键包？

一键包是一个包含所有必要文件和依赖的压缩包，用户只需解压后双击启动脚本即可使用，无需安装Python或其他任何依赖。

## 优势

- **免安装**：无需安装Python环境
- **可移植**：可复制到任意位置使用
- **方便迁移**：可轻松复制到其他电脑
- **版本隔离**：不会与系统Python冲突

## 制作步骤（详细流程）

### 第1步：准备工作

- **操作系统**：Windows 10/11
- **网络连接**：需要下载Python和依赖包
- **存储空间**：至少需要500MB空间

### 第2步：下载项目代码

```bash
git clone https://github.com/ops120/bai-piao-master
cd bai-piao-master
```

### 第3步：下载便携式Python

1. 访问 [Python官网](https://www.python.org/downloads/windows/)
2. 下载 **Windows embeddable package (64-bit)** 版本
   - 推荐使用 Python 3.12.x 版本
   - 例如：`python-3.12.0-embed-amd64.zip`
3. 在项目根目录创建 `portable-python` 文件夹
4. 解压所有文件到 `portable-python` 目录

### 第4步：配置便携式Python

1. 编辑 `portable-python/python312._pth` 文件
2. 找到 `import site` 这一行
3. 取消这一行的注释（删除前面的 `#`）

**修改前：**
```text
python312.dll
python3.dll

# Uncomment to enable site-packages
#import site
```

**修改后：**
```text
python312.dll
python3.dll

# Uncomment to enable site-packages
import site
```

### 第5步：安装pip

1. 下载 `get-pip.py` 脚本：
   ```bash
   Invoke-WebRequest -Uri "https://bootstrap.pypa.io/get-pip.py" -OutFile "portable-python/get-pip.py"
   ```

2. 安装pip：
   ```bash
   & "portable-python/python.exe" "portable-python/get-pip.py"
   ```

3. 安装完成后删除 `get-pip.py`：
   ```bash
   Remove-Item "portable-python/get-pip.py"
   ```

### 第6步：安装项目依赖（关键步骤）

必须使用 `--target` 参数将依赖安装到 portable-python 目录：

```bash
& "portable-python/Scripts/pip.exe" install gradio requests Pillow anyio fastapi uvicorn httpx --target "portable-python/Lib/site-packages"
```

**重要说明：**
- Gradio 4.0+ 版本依赖 many other packages，包括：fastapi、uvicorn、httpx、anyio等
- 必须使用 `--target` 参数确保依赖被安装到 portable-python 目录
- 使用清华源加速下载：`-i https://pypi.tuna.tsinghua.edu.cn/simple`

### 第7步：创建必要的目录和配置文件

1. 创建必要的目录：
   ```bash
   mkdir conf
   mkdir logs
   mkdir outputs
   mkdir templates
   mkdir docs
   ```

2. 确保 `conf/moark_config.example.json` 存在

### 第8步：创建启动脚本

创建 `启动-ANSI.bat` 文件（使用英文避免编码问题）：

```batch
@echo off

rem 白嫖大师 - 一键启动脚本
rem 作者：你们喜爱的老王

rem 切换到脚本所在目录
cd /d "%~dp0"
echo Current directory: %CD%

rem 检查便携式Python是否存在
if not exist "portable-python\python.exe" (
    echo Error: Portable Python not found, please check installation
    pause
    exit /b 1
)

rem 创建logs目录（如果不存在）
mkdir logs 2>nul
echo Ensured logs directory exists

rem 启动应用
echo Starting 白嫖大师...
echo Please wait, first startup may take some time...
echo Command: "portable-python\python.exe" "moark_image_edit_ui.py"

"portable-python\python.exe" "moark_image_edit_ui.py"

rem 如果应用意外退出，显示提示
if errorlevel 1 (
    echo Application startup failed, please check error messages
    pause
    exit /b 1
)

rem 正常退出
echo Application started successfully!
echo Please open browser and visit: http://localhost:11111
echo Press any key to close window...
pause >nul
```

### 第9步：测试运行

1. 双击 `启动-ANSI.bat` 脚本
2. 等待应用启动
3. 打开浏览器访问 `http://localhost:11111`
4. 确认应用正常运行

### 第10步：创建压缩包

使用PowerShell创建一键包：

```bash
Compress-Archive -Path "portable-python", "moark_image_edit_ui.py", "启动-ANSI.bat", "conf", "requirements.txt", "一键包制作指南.md" -DestinationPath "白嫖大师-一键包.zip" -Force
```

## 一键包包含文件

一键包应包含以下文件和目录：

| 文件/目录 | 说明 |
|-----------|------|
| `portable-python/` | 完整的便携式Python环境（含所有依赖） |
| `moark_image_edit_ui.py` | 主程序文件 |
| `启动-ANSI.bat` | 一键启动脚本 |
| `conf/` | 配置目录（包含示例配置） |
| `requirements.txt` | 依赖列表文件 |
| `一键包制作指南.md` | 使用说明文档 |

## 常见问题与解决

### 问题1：启动脚本报错"不是内部或外部命令"

**原因**：中文编码导致乱码

**解决方法**：使用英文提示的启动脚本（`启动-ANSI.bat`）

---

### 问题2：ModuleNotFoundError: No module named 'anyio'

**原因**：anyio模块未安装到portable-python目录

**解决方法**：
```bash
& "portable-python/Scripts/pip.exe" install anyio --target "portable-python/Lib/site-packages"
```

---

### 问题3：ModuleNotFoundError: No module named 'fastapi'

**原因**：fastapi模块未安装到portable-python目录

**解决方法**：
```bash
& "portable-python/Scripts/pip.exe" install fastapi --target "portable-python/Lib/site-packages"
```

---

### 问题4：端口11111被占用

**解决方法**：在启动脚本中添加端口检测和进程终止代码，或更换端口

---

### 问题5：选项卡不显示或功能缺失

**原因**：DEFAULT_CONFIG缺少异步模型配置

**解决方法**：检查并添加以下配置到 `moark_image_edit_ui.py`：

```python
DEFAULT_CONFIG = {
    # ... 其他配置 ...
    "async_txt2img_models": [
        "FLUX.1-dev",
        "LongCat-Image",
        "flux-1-schnell",
        "Qwen-Image-2512",
        "Z-Image",
        "Qwen-Image"
    ],
    "async_edit_models": [
        "LongCat-Image-Edit",
        "Qwen-Image-Edit-2511",
        "FLUX.1-Kontext-dev"
    ]
}
```

---

### 问题6：gr.Dataframe报错 max_height 参数

**原因**：Gradio版本不兼容

**解决方法**：移除 `gr.Dataframe()` 中的 `max_height` 参数

## 注意事项

1. **存储空间**：便携式Python和依赖包会占用约400-500MB空间
2. **首次启动**：首次启动可能需要10-30秒加载依赖
3. **网络连接**：使用时需要网络连接以访问Moark API
4. **API Key**：用户需要从Moark官网获取API Key
5. **防火墙**：首次运行可能需要允许应用访问网络

---

## 版本更新

当项目更新时，需要重新制作一键包：

1. 拉取最新代码：`git pull`
2. 更新依赖到portable-python目录：
   ```bash
   & "portable-python/Scripts/pip.exe" install --upgrade gradio requests Pillow anyio fastapi uvicorn httpx --target "portable-python/Lib/site-packages"
   ```
3. 重新创建压缩包
